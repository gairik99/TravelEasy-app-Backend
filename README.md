# Easy Life

THIS IS TRAVELEASY APP BACKEND

## APIS
- /api/hoteldata(POST)<br/>
All hoteldata added to database

- /api/categorydata(POST)
All category data added to database

- /api/hotels(GET)
Get all hotels data

- /api/category(GET)
Get categorywise hotel data

- /api/hotels/:id(GET)
Get a specific hotel data

- /api/wishlist(GET)
Get wishlist data

- /api/auth(GET & POST)
register and signup
